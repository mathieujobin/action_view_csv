# ActionViewCsv
===============

ActionView template handlers for converting csv into an html table

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'action_view_csv'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install action_view_csv
